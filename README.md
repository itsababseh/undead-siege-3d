# UNDEAD SIEGE 3D 🧟

A 3D first-person zombie survival game built with Three.js. Inspired by Call of Duty Zombies.

## 🎮 [PLAY NOW](https://itsababseh.github.io/undead-siege-3d/)

## Features
- **Full 3D FPS** — Three.js powered with textured environments, dynamic lighting, and fog
- **3 Weapons** — M1911 (starter), MP40 (wall buy), Shotgun (wall buy)
- **4 Perks** — Juggernog, Speed Cola, Double Tap, Quick Revive (permanent until death)
- **Boss Zombies** — Every 5th round features a boss + 15% elite zombie chance
- **Progressive Difficulty** — CoD-style tier jumps every 5 rounds
- **Buyable Doors** — West Wing & East Chamber unlock new areas + more zombies
- **Leaderboard** — Local high scores (top 10)
- **Audio System** — Eerie background music, zombie screams, weapon SFX (Web Audio API)
- **Minimap** — Real-time tactical overview
- **Mobile Support** — Touch controls with virtual joystick
- **Pause System** — ESC to pause, click to resume

## Controls
| Action | Desktop | Mobile |
|--------|---------|--------|
| Move | WASD | Virtual Joystick |
| Look | Mouse | Touch Drag |
| Shoot | Left Click | Fire Button |
| Reload | R | R Button |
| Buy/Interact | E | E Button |
| Switch Weapons | 1-3 | Weapon Buttons |
| Pause | ESC | — |

## Tech Stack
- Three.js 0.162 (3D rendering)
- PointerLockControls (FPS camera)
- Web Audio API (synthesized audio)
- Procedural textures (no external assets)
- Single HTML file, zero dependencies
